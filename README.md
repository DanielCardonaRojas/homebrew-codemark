# codemark Homebrew Tap

This is the Homebrew tap for [codemark](https://github.com/DanielCardonaRojas/codemark).

## Installation

```bash
brew tap DanielCardonaRojas/codemark
brew install codemark
```

## Upgrading

```bash
brew upgrade codemark
```

## Uninstalling

```bash
brew uninstall codemark
brew untap DanielCardonaRojas/codemark
```
